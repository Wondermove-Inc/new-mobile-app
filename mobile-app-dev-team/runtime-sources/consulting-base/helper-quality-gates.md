# Helper Quality Gates

## Gate 1: File intake readiness

Pass requires full target files by file transfer or documented fallback task
attachment. Missing files are reported as missing evidence.

## Gate 2: Role and boundary clarity

Pass requires clear role scope, must-not-do boundaries, reporting route, and
human-gate boundary in the target Agent's operating files.

## Gate 3: Review loop clarity

Pass requires the target process to preserve:

target plan -> leader plan review -> target work -> result report -> leader
result review -> feedback incorporated -> completion only after approval.

## Gate 4: Tracking readiness

Pass requires Workboard guard for consulting follow-through and wake guard for
long-running, waiting, delegated, or review-pending work.

## Gate 5: Evidence readiness

Pass requires source files, reviewed artifacts, missing evidence, and completion
proof to be recorded in a receiver-accessible place.

## Gate 6: Skill candidate routing

Pass requires durable skill candidates to route through Tasks, leader review,
user approval when needed, and approved skill lifecycle. Helper must not imply
it can promote live skills directly.

## Gate 7: Human-gate preservation

Helper reports evidence readiness and process compliance only. Helper does not
approve human-gated risk, production, release, legal, compliance, credentials,
external messaging, failed-gate risk, or live skill promotion.
