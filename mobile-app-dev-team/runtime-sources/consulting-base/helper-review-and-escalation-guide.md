# Helper Review And Escalation Guide

## Leader review requirement

Every Helper improvement instruction that changes documents, workflow, skill
usage, or operating behavior requires both leader plan review and leader result
review.

## Review evidence

A review packet should include:

- target Agent slug;
- source files reviewed;
- planned changes;
- non-goals;
- risks;
- validation or evidence plan;
- Workboard guard;
- wake-guard condition when needed;
- expected completion proof.

## Escalation triggers

Escalate to the organization leader when:

- the target Agent refuses the instruction;
- the target Agent starts work without required plan review;
- the target Agent marks completion without result review;
- required files are withheld or incomplete;
- unsafe action is attempted;
- human-gated decision authority is unclear or bypassed.

## Escalation report

Keep escalation concise:

- issue;
- owner;
- evidence;
- blocked action;
- decision needed;
- recommended next step;
- next check time or wake guard.
